USER MANUAL

IdentaMaster security functions are managed through the Administrative Module by running Windows command:

Start -> Programs -> IdentaMaster

With Administrative module you will perform the following operations:
  • Activating and Terminating user’s Windows account with Biometric Capabilities.
  • Choosing security options for access to computer
  • Collecting log data on PC access
  • Verifying the fingerprint reading device status
  • Enter/Update IdentaMaster license value to enable various biometric hardware


Product activation

Scanner Testing

Select Tab ‘Test Scanner’ to bring the device control pane. Depends on biometric scanning devices currently enabled to use with the given license, you may execute scanner testing procedure. Just select particular device and you will be prompted to place finger on the scanner. If scanner is not connected to your PC corresponding message will be issued. If scanner is properly connected and it reading your fingerprint – image will be rendered on the black panel.



User Enrollment

Enrollment procedure with fingerprint scanners

Select ‘User Enrollment’ tab if you are ready to register biometric PC access mode for Window’s user. You will be presented with the windows account listing. Just click or double click on the record and you will be prompted to activate or deactivate/update current user (if he/she already been registered).



Choosing a current activated user, the selection prompt let you choose the security mode from the following options:
  • Password or Biometric
  • Biometric Only
  • Password Only

To avoid problem at the beginning, we strongly recommend selecting mode Password or Biometric. It will allow time to learn how to properly use biometrics to be successfully identified at the same time you have an alternate regular way to logon with your password. After while, if you accustomed yourself easily logon using biometric scanner, you may establish advanced security mode ‘Biometric Only’. You will be also asked to enter and confirm windows account password to ensure that the particular user would be linked to created biometric credential record.

Note: If you change windows account password, you must confirm it again on User Enrollment tab by clicking on user name and reentering new password as prompted.

From this point you may proceed to the fingerprint enrollment.

  • Select Biometric Provider from the drop-down list above
  • Select finger you would like to enroll
  • You will be prompted to place finger on the scanner surface. Depends biometric Provider you are required to place, lift and place finger few times until good biometric data would be collected.

Follow the rules listed below:
  1. Place the center of the finger (core) tip in the middle of the sensor – there is a better chance to be accepted.
  2. Cover the sensor area completely with the finger to ensure the maximum fingerprint surface contact area – it will collect more information (i.e. minutiae)
  3. Use medium pressure on the sensor – too heavy or light pressure may result in a poor image and consequently incorrect enrollment.
  4. Keep fingerprint rotation minimum during the enrollment and use the same orientation ( unless instructed by particular Provider)
  5. It is best not to have very wet or very dry finger during enrollment procedure. Wipe your finger with cloth or paper towel if it wet, or, otherwise, if it too dry – moisturize the finger by breathing on it.
    The image quality could be improved tremendousely by taking care of the dry/wet fingerprint condition

During the enrollment procedure monitoring cursor on top of the page will be indicating progress. At the end of the enrollment cycle message will be popped up: “Successfully registered”. You also may cancel the enrollment process at any time.

Logon to your Windows account

After successful enrollment you may close the Administrative console and log off from the current account. Upon logon, you will be displayed with the following interface



Each active IdentaMaster user being represented with IZ logo. User clicks on the icon with his/her account name and start verification procedure: depends on security mode it should be Biometric, Password, or either (Biometric or Password) verification. On top of the screen you may see Biometric Device Status image, such as "Online" or "Error". If status indicates "Error", reconnect device and press link "Refresh".

Separately, you may see Biometric Logon icon that follow you to Identification (1:n) mode. Please click on it and perform Biometric Identification procedure – upon successful ending Windows will load corresponding profile.
